Introduction

In this video you will learn what the DieBenutzerumgebung really is and how it is intended to be used. There will be some rough edges and bugs but it should be useable as a daily driver. It at least does the job for me on all my linux devices. However, maybe you have some different workflows, other requirements for your tasks.

So if you feel that you have suggestions/criticism/ideas/opinions I'd like to hear about them.

In the examples I have a tool running that displays my keystrokes, so you can see what I typed to achieve the result.
